📖 Sobre a Pokédex

Esta Pokédex é um projeto interativo que utiliza a API DiegoOilv para trazer informações detalhadas do mundo Pokémon.

🔎 Pokémon

Pesquise por ID ou Nome para ver:

  • 🖼️ Sprite, imagem oficial e miniatura
  • 📊 Status base (HP, Attack, Defense...)
  • 🌱 Tipo(s), evolução e descrição
  • ⚖️ Altura, peso, gênero e habilidades

⚔️ Move

Busque por ID ou Nome e descubra:

  • 🎯 Precisão e poder
  • 💥 Categoria
  • 🔠 Nome em inglês, japonês e chinês

🎒 Item

Pesquise por ID ou Nome para ver:

  • 🧺 Tipo de item
  • 📜 Descrição do efeito
  • 🌍 Nome em vários idiomas

👻 Type

Busque apenas por Nome para visualizar:

  • ✔️ Tipos em que é eficaz
  • ⚠️ Tipos em que é ineficaz
  • ❌ Tipos sem efeito
